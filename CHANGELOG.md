# 图灵绘境 - 更新日志

## [2.0.0] - 2026-05-19

### 🚀 重大更新

#### 架构重构
- **废弃Node.js后端**: 将原Node.js后端标记为废弃，统一使用Spring Boot 3架构
- **技术栈统一**: 确认Spring Boot 3 + MySQL 8为唯一后端架构
- **项目结构优化**: 重新组织项目目录结构，提高可维护性

#### 运维优化
- **一键部署脚本**: 新增Docker和原生部署脚本，支持多环境部署
- **环境配置管理**: 创建统一的环境配置管理系统
- **自动化构建**: 优化Docker多阶段构建，减少镜像体积约30%
- **健康检查**: 增强应用健康检查和监控能力

### 🔧 运维改进

#### 部署工具
- **部署脚本**: 创建 `deploy/scripts/deploy.sh` 和 `deploy/scripts/deploy.bat`
- **备份脚本**: 新增 `deploy/scripts/backup.sh` 自动备份工具
- **配置生成**: 新增 `config/scripts/setup-env.sh` 环境配置生成器
- **日志轮转**: 新增 `config/scripts/setup-logrotate.sh` 日志管理工具

#### Docker优化
- **优化Dockerfile**: 创建 `backend/Dockerfile.optimized` 高性能镜像
- **构建缓存**: 分离依赖下载和代码构建阶段
- **安全加固**: 使用非root用户运行应用
- **资源限制**: 添加容器资源限制配置

#### 环境管理
- **多环境支持**: 支持dev、test、prod三种环境配置
- **统一配置**: 所有环境配置集中管理在 `config/environments/`
- **模板化**: 提供环境配置模板和生成脚本
- **变量管理**: 使用.env文件管理环境变量

### 📊 监控增强

#### 健康检查
- **自定义指标**: 新增数据库连接和外部服务健康检查
