## 22:53 首页渐变去掉「AI味」，换成暖桃珊瑚系

**背景**：蓝紫渐变是 ChatGPT/Claude/Midjourney 等 AI 产品的制服色，不符合"创作工具"定位。

**方案**：基于「既梦 + 美图设计室」的浅色/清新/高级感，全量替换 index.vue 为暖桃珊瑚系：
- `$primary`: #8B9DC8 → #D4957A（暖桃珊瑚）
- `$primary-grad`: #D4957A→#C9826E
- `$secondary`: #C4B5E0 → #F0D8C8（暖粉米）
- `$accent`: #A3B8A5 → #B4C4AC（柔鼠尾草）

**改动量**：16 处替换，覆盖 SCSS 变量、背景纹理、骨架屏、Hero 光晕、按钮/图标/标签/封面渐变。蓝紫引用 0 残留，SCSS 编译 0 错误。

## 23:19 修复 mine.vue onShow 导入错误

**错误**：`mine.vue:412 SyntaxError: @dcloudio/uni-h5-vue 不导出 onShow`

**根因**：`import { ref, computed, onMounted, onShow } from 'vue'` — UniApp Vue3 中 onShow 是小程序/H5 生命周期，属于 @dcloudio/uni-app，不是 Vue 核心 API。

**修复**：
```js
// 改前
import { ref, computed, onMounted, onShow } from 'vue'

// 改后
import { ref, computed, onMounted } from 'vue'
import { onShow } from '@dcloudio/uni-app'
```

**扫描**：前端全部 .vue 文件无其他同类问题。验证通过。
