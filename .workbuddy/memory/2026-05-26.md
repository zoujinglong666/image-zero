## 修复头像不显示（H5 + 微信小程序均无效）（23:15）

### 根因
`u-avatar` 组件源码（`node_modules/uview-pro/components/u-avatar/u-avatar.vue` 第7行）：
```html
<image v-if="!uText && avatar" :src="avatar" />
<text v-else-if="uText">{{ uText }}</text>
```
其中 `uText = String(props.text)[0]`。只要传了 `text` prop 且非空，`uText` 就是 truthy，`!uText` 永远 false，**图片永不渲染**。无论 `src` 有没有值，`text` 优先级最高。

`mine.vue` 始终传 `:text="avatarText"`（至少是 `'?'`），所以 `u-avatar` 永远只显示文字，不显示图片。后端接口返回的 avatarUrl 完全正确但组件不渲染。

### 修复（3处）
| 行号 | 修复内容 |
|------|---------|
| 21 | 顶部用户信息卡片的 u-avatar：`:text="userStore.avatarUrl ? '' : avatarText"` |
| 280 | 微信小程序头像编辑按钮的 u-avatar：同上 |
| 295 | H5 头像编辑的 u-avatar：同上 |

**逻辑**：有 avatarUrl 时传空字符串（`''`）= falsy → 组件走图片渲染；没有时传 `avatarText` → 走文字兜底。
